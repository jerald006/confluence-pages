Create Booking

Only the users that have their role as ota can create and settle bookings on the platform.

The bookings can be created on the platform by calling the Create Booking API (/booking/create) with the following details:

  • _id → The Unique ID of the booking on the OTAs system

  • fare → The breakup of the fair along with the currency and total

  • booking_type → Hotel, Flight, etc.

  • start → Booking start date

  • end → Booking end date

  • cancellationPolicy → if any

  • isRefundable → Whether the booking is refundable or not

  • bookingDetails → Any information that needs to be stored on the blockchain for the booking

Pre-requisites

The create booking API requires a payment to be made for the specified amount for the given booking on ConnexPay.

So, a sale has to be generated for the same _id as the create booking.

Settle Booking

The ota that made a booking can settle the booking. When that happens, the tokens will be transferred to the correct parties.

Pre-requisistes

A booking with the given _id has to be created first.
