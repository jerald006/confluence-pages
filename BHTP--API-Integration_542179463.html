XeniApp : BHTP- API Integration

Created by User d8ec2, last modified on Jun 09, 2021

Technology Notes

  • The API is accessed over HTTPS

  • Interactions are via REST

  • All requests are to be sent in JSON format

  • All requests must be sent using TLS 1.2

Suggested Tools

  • POSTman

    : A free standalone request/response REST client allowing you to test API interactions prior to writing code. It also allows for you to save requests for later use

Standards

  • All dates follow ISO 8601 standard

  • All countries and states follow ISO 3166-2 standard

  • All flight information is identified by IATA codes
