I created a Bug template in Jira called Bug Template which automatically populates the table and details must be manually filled.

  1. Click on ‘Create’

  2. Project defaults to Xeniapp, Issue type should be Bug

  3. Add a brief summary of the bug. Do not add anything in the description.

  4. Select Create on bottom of the screen

  5. Once the bug is created, click on the bug link and click the … right hand side top corner of the page and select Apply template

7. Click on apply of the first template - ‘Bug Template’ and hit submit.

8. It will take a moment and create a bug issue with the template.

9. You can fill the details and other fields like attachments, components, linked issues, priority, labels, etc. and save the page.
