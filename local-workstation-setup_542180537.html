XeniApp : local workstation setup

Created by Rajesh, last modified on Jan 14, 2022

Repos

Required Repos

  1. xeniapp-react-3.0: https://github.com/xeni-app/xeniapp-react-3.0

  2. xeniapp-admin-dashboard-3.0 (coming soon)

  3. xeniapp-component-library: https://github.com/xeni-app/xeni-component-library

  4. falcon-components: https://github.com/xeni-app/Xeni-falcon-components

Brief overview of the repos

xeniapp-react-3.0 is a client facing application which provides customers ability to book their flights, hotels, cars and other travel related requirements.

xeniapp-admin-dashboard-3.0 is a platform for customer service reps to address any issues reported by clients

xeniapp-component-library, as the name suggests hold react components which would be used by xeni applications, so the look and feel of the components would be same across

falcon-components(short-lived) is a react components repo which holds the use-cases and other flows with static data, this would be used only for reference purposes

Setup

NVM

  1. Download nvm or install nodejs 16

  2. open your terminal

  3. nvm install v16.5.0

  4. nvm alias default v16.5.0 (this will set you to node version 16)

Component library

  1. cd xeniapp-component-library

  2. npm install

  3. npm link

    1. this will create a symbolic link of the component-library in your local node

    2. when changes are pushed to component-library a regular pull would be sufficient rather than installing node-modules

Xeniapp react

Before starting the xeniapp-react, we will get the npmjs token

  1. open https://www.npmjs.com/

  2. login using your username and password

  3. on top right corner, click your avatar

  4. click on access tokens

  5. check you are part of xeniapp orgnanization

    1. note: if you are not part of xeniapp organization get help of vaibhav (Unlicensed) to get you added

  6. click on generate new token

  7. enter the details and click on generate

  8. copy the token

    1. note: this token is shown only once so make sure to save it

Setting up the workspace

  1. cd xeniapp-react-3.0

  2. cp .npmrc-ci .npmrc

  3. vi .npmrc

  4. replace ${NPM_TOKEN_READ_ONLY} with the token you copied from npmjs

  5. npm install

  6. npm link @xeniapp/xeni-component-library

note: under node_modules for xeni-component-library you should see the small arrow indicating its linked now
