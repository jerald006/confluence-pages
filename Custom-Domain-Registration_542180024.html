XeniApp : Custom Domain Registration

Created by vaibhav (Unlicensed), last modified on Mar 30, 2022

Step 1 - Agency Command Center - Ask the respective agency owner to do this step

  1. Login to the command center, go to settings, edit settings and under the domain registration, specify the custom domain - https://prnt.sc/Kob2gMSbyNEK.

Step 2 - Xeniapp Team - Certificate issue

  1. Login to AWS. Go to https://us-east-1.console.aws.amazon.com/acm/home?region=us-east-1#/certificates/request and request a new certificate - https://prnt.sc/5wbmPdvqbW0F . Refer to point 2 to all the qualified domain names.

  2. To get the name for fully qualified domain name for the new certification, look for a previous certificate which has store1.myxeni.com as domain name and in use=”Yes” - https://prnt.sc/CAs_F6XEetUK . Paste all the domains under this certificate while requesting the new certificate.

  3. Enter all the domain names and keep the validation method as DNS validation - https://prnt.sc/pWKcytsY03qe . Click on request.

  4. Click on the certificate in 5 mins - it should show Success for all the previous domain, for the new domain it should show pending validation - https://prnt.sc/Znvt2zNZRfVH

Step 3 - Domain Manager like Godaddy - Agency Owner

  1. Go the domain manager and add a CNAME record. Value can be found under the certificate - https://prnt.sc/Znvt2zNZRfVH

  2. Add another CNAME - Name - www , value - store1.myxeni.com.

  3. Forward https://yourdomainname.com to https://www.yourdomainname.com

Step 4 - Xeniapp Team - CloudFront Distribution

  1. Once above is done, all the domain should show as as Success under the certificate - https://prnt.sc/k_Lo-fJqFbKq

  2. Go to Cloudfront distribution which serves store1.myxeni.com and edit it to add the new website - https://prnt.sc/tysL24J_i5Q2

  3. Select the new certificate from the Custom SSL dropdown - https://prnt.sc/wgELWiA1YLvf

  4. Click Save Changes

  5. The In use status of the newly issues certificate should change to Yes.

  6. Verify all the domains

Step 5 - Xeniapp Team - Terraform

  1. Login to Xeniapp’s terrform

  2. Choose xeniapp-infra-frontend-prod and click on variables

  3. Edit the storefront variable and add the newly added domain name to it.

  4. Click on Runs and new Plan. Add the reason for the new plan - https://prnt.sc/CyGllPH08Ob8

  5. Confirm and apply
