Deposit

POST /deposit/crypto

Make crypto payments to Xeni.

GET /deposit/crypto

Get status of Crypto Payment made in Xeni by the ID.

Withdraw

POST /withdraw/fiat

Using this endpoint, the user can withdraw their XeniTokens and get it converted to Fiat.

To do that, applications will send a POST request along with the necessary details such as:

  • Bank Details

  • BookingID

  • Amount

After providing the details, these will be processed by the payment service providers.

  1. Validations

  • All bank fields need to be validated

GET /withdraw/fiat

The user can also get the status of their Withdraw request if they want.

This will return the entire object that contains the withdraw request.
