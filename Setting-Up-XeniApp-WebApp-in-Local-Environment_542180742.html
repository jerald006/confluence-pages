 2.0 Installation Steps

2.1 Installation of MongoDb

 Steps:

  1. Download the Software from the following link.

    1. https://www.mongodb.com/try/download/community

    2. Screenshot for reference

2. For installation use the below link

https://www.youtube.com/watch?v=FwMwO8pXfq0

3. Download the 2 DB dump file from the drive and extract the zip file.

4. Rename the biirdee_db extracted folder to “biirdee_prod_db”.

5. Open the command prompt and run the following script 2 commands after the successful execution of the first command.

mongorestore -d biirdee_prod_db "fullpath\biirdee_prod_db"
mongod --port 27017 -dbpath "fullpath\biirdee_prod_db"

6. Rename the xeni_db extracted folder to “xeni_prod_db”.

7. Open the command prompt and run the following script 2 commands after the successful execution of the first command.

mongorestore -d xeni_prod_db " fullpath\xeni_prod_db"
mongod --port 27017 --dbpath " fullpath\xeni_prod_db"

 8. After the successful export of the dump file, the collections can be verified in the MongoDb Compass.

2.2 Installation of Redis

Steps:

  1. Download the Software from the following link.

o   https://download.redis.io/releases

o   Screenshot for reference

2. Open the installed Redis folder in C:\Program Files

3. Double click “redis-server” file in the folder.

4. To verify Redis is running, double click “redis-cli” file.

5. Type the command ping and it should respond with PONG.

6. Please review the screenshot for reference.

2.3 Running the Front End Code

Steps:

1. Please get access to Github repository.

2. Create personal token in Github setting.

3. Clone the repository using the following command. 

C:\>git clone “Repositoryurl”

4. Provide the Git credentials on the Prompt.

5. After the successful download command prompt will show up.

6. Run the following command to install the yarn globally.

C:\>npm install –g yarn

7. Run the following command to install the packages. 

C:\>yarn

8. Command to run the Frontend Application.

C:\>yarn start

2.4 Running the Back End Code

Steps:

1. Follow the steps from 1 to 5 of 2.3

2. Checkout to the branch “agency-subscription-flow” using following command.

C:\>git checkout agency-subscription-flow

3. Run the following command to the run the 30 script files sequentially one after the other without skipping the number.

C:\>git checkout agency-subscription-flow
..
..
C:\>node src/migration/ 30_split_full_name 

4. Run the following command to install the nodemon globally.

C:\>npm install –g nodemon

5. Run the following command to install the packages.

C:\>npm install 

6. Command to run the Backend Application.

C:\> nodemon server.js

2.5 Steps to Host XeniApp

Steps:

1.Open the “hosts” file in the location “C:\Windows\System32\drivers\etc”.

2. Make the following changes.

127.0.0.1    xeniapp.com
127.0.0.1    name.xeniapp.com

3. After the application is hosted successfully.

 3.0 Steps to Merge the Developer Code to Master

3.1 Create a PR

  1. Please get the latest pull from the Master branch for the corresponding branch that relates to the task.

  2. Create a new branch for the task with task-id as branch name, please review the screenshot for reference.

3. After the making the code changes, please save the changes with git add --all command. Please review the screenshot for reference.

4. Please commit the changes with appending ticket id in the end git commit am "Message [XEN-Id]". Please review review the screenshot for reference.

5. Please push the changes to GitHub with the command git push origin branchName . Please review the screenshot for reference.

6. Login to Github, Create PR and Raise a request to Review.
