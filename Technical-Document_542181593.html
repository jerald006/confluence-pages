Code Structure

Package description

  1. Config : In this package all the environment variable are fetched and loaded to run the module.

  2. grpcHandler : This package holds gRPC request calls for all the internal modules.

  3. httpHandler : This package holds the handler functions to make the gRPC calls through HTTP requests.

  4. httpServer : This package holds all the necessary HTTP end points which will be exposed to end user.

  5. .env and sample.env : These files consists of all the necessary variables for the module.

# Consul
CONSUL_ADDRESS=localhost:8500

# Go Micro
MICRO_BITPAY_SERVICE=bitpay
MICRO_NAME=xenipayment-gateway
MICRO_REGISTER_INTERVAL=3

# HTTP
HTTP_ADDRESS=localhost
HTTP_PORT=6969

Following are the variables which are used to run xenipayment-gateway.
