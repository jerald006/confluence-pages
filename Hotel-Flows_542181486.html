End User Hotel Booking

From landing page, hotel bookings can be made by navigating to Book Now to Hotels tab.

Search Hotels

  • User can search hotel by

    • entering destination location in “Going to” field

    • selecting dates from Date pick field

      • User can select multiple days by clicking on start date and last date of booking

        • Eg. If user want to select from 12 February to 18 February, selecting 12 and 18 includes all dates between them

    • choose rooms and guests from Rooms and Guest dropdown

      • User can select multiple rooms by selecting Add Room. Occupancy of each room is four adults and four children. Use has to click on Done after choosing rooms as per guests

Hotel Details

  • After entering booking details, clicking Search button results available properties as per search criteria in a new page.

  • User has Go to Map and other filter options to choose properties as per needs

    • Exclusive Deals

    • Hotel Name

    • Category

    • Amenities

  • User can click on Add to Compare button and add properties to compare

  • User can filter properties using price filter

  • Clicking on Details button of a particular property routes to respective hotel details page, where hotel description, details of rooms, location, amenities and policies are described.

Book Now

  • Rooms section in hotel details page consists of list of available rooms with cancelation policy, Extras, price details.

  • Clicking on Book now button of chosen room routes to booking page. It consists of booking information, check-in information, room details, price details, guest details which has to be entered by user and Billing details.

Payment can be done using two methods

XeniPay for paying with cryptocurrencies

  1. Select XeniPay radio button and proceed with Complete Booking

  2. User has to select respective wallet and cryptocurrency.

  3. User has to confirm the transaction from wallet.

  4. Upon successful completion of transaction, user will be routed to billing page saying ”Processing Your Payment”. Same details can be found in user’s My Trips page with status “Awaiting Payment“. If transaction fails or session expires then status will be “Failed”.

  5. Once payment is successful, status of the booking with changes to “Confirmed” automatically. Same details can be found on user’s My Trips page for the booking with the Confirmation Booking ID.

  6. User can cancel the booking by clicking Cancel Booking from the action button of the booking. This changes the status of the booking to “Cancelled and cancellation ID is generated”

Debit/Credit Card for paying with fiat currencies.

  1. Select Debit/Credit radio button and proceed with Complete Booking

  2. User has to select respective account details and confirm the payment.

  3. Once payment is successful, status of the booking with changes to “Confirmed” automatically. Same details can be found on user’s My Trips page for the booking with the Confirmation Booking ID.

  4. User can cancel the booking by clicking Cancel Booking from the action button of the booking. This changes the status of the booking to “Cancelled” and cancellation ID is generated.
