XeniApp : Agency API access

Created by Rajesh, last modified on May 03, 2024

Some of the API endpoints are exposed for external use by agencies. To achieve this is a two-step process

  1. The agency owner needs to contact the Xeni helpdesk to get the API key

  2. The agency needs to add a custom header when calling the API's

Actions required to be taken by the Xeni helpdesk

Login to the admin portal

Navigate to manage agencies

Click on gear icon for the selected agency

Enable “Allow agency to call api“

Provide the agency owner with the key listed below

Note: every time you disable and enable the key changes

Instructions to agency when making the API calls

Two pieces of information would be needed when making the API calls

  1. Custom header “x-xeni-token“ with the key provided by helpdesk

  2. Request body with “agencyName” with your agency name

List of API's exposed to agencies as of (May-03-2024)

https://app.swaggerhub.com/apis/xeni/Xeni-api/0.1.0 - use admin@xeni.com to make necessary changes
