Functional objective

Once the user selects the grade type and the selected cabin, it is required to process the booking and payment of the item. For this purpose, the flow should be:

  • BasketAdd - The selected identified item should be added to the vendor´s basket

  • Expose a screen for travelers to select the dining details

  • Present to the travelers the personal information form to enter details

  • create the booking on the vendor side with the payment details. (/book.pl)

  • Process the agency markup debit (Xeni payment gateway).

  • Present to the traveler the final screen of the booked detailed item.

  • Booking receipt.

Context

The user has already selected the cruise item to consume.

Proposal Solution

Cruise booking flow

Functional Description

Add item to vendor basket

The vendor offer an endpoint to add a selected item into the basket with a 14 minutes on Hold (No Deposit, no guarantee) in order to process the payment and confirm the booking.

POST: /basketadd.pl
curl --location 'https://fusionapi.traveltek.net/2.1/json/basketadd.pl?requestid=10147191me09e-48c2-90c8-20322bbf185f&sessionkey=C9F4FF82-6A90n42C1-8206-F0CF4E9790E1&resultkey=null&type=cruise&resultno=212_0.1058090091&gradeno=212%3APGO%3A2&cabinresult=212%3APGO%3A2_0&ratecode=PGO&site=xeni.site.traveltek.net&currency=USD&sid=49572' \
--header 'Accept: application/json; charset=utf-8'

Params:

  • requestid: Authorization accessToken

  • sessionkey: The session key started by consumer, returned in the response body of /cruisecabin.pl

  • resultkey: Value returned from /cruisecabingrades.pl

  • type: always "cruise"

  • resultno: Returned in the response of /cruisecabin.pl

  • gradeno: The selected grade number returned on the /cruisecabin.pl as Criteria.

  • cabinresult: Returned in the selected option from /cruisecabin.pl

  • ratecode: Returned from the selected option of /cruisecabin.pl

  • site: From vault - The Xeni site in the vendor

  • SID: From vault - The SiteId of XENI in the vendor config.

  • currency: The display preferred currency

And received a successful HTTP:200 without any error description on the “Errors” object.

Remove Item from Basket:
