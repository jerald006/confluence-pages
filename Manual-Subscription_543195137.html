XeniApp : Manual Subscription

Created by User 7e8c3 on Dec 02, 2022

  • The agent will fill up the first part of the subscription i.e., the agency sign up

  • The agent will leave the payment page blank

  • The agency entry will appear on the admin panel

  • Admin needs to click on the subscription checkbox for that agency and it will automatically display Manually Subscribed

  • We will provide the agent with the URL of their agency and the agent will be able to log in with the credentials entered on the agency sign-up page

  • Agency will be in the free trial period as long as the admin wants to give the free trial

  • There will be no entry of subscription of that agency on stripe but agency booking will be reflected in stripe

  • Once admin decides to end the free trial for this agency, an email needs to send to the agent that the free trial is ending soon and in order to continue using the agency please add the credit card details in the command center – payment setting

  • Once the agent adds the credit card details, the admin subscription will be the auto subscription like other subscriptions

  • Manual subscribed will be changed to active and the billing period will get reflected in the admin panel

  • In stripe, we will see that agency’s subscription entry and all the details

  • If the agent decides not to extend the subscription, the admin will disable the agency from the admin panel

  • The manual subscription is valid until Dec 25th (the manual subscription checkbox will not be there on the admin panel)
