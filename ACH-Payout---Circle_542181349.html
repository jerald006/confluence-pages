Plaid Integration Guide

To use Circle for ACH transfers, Plaid is required. To add Circle to a Plaid account, do the following.

https://plaid.com/docs/auth/partnerships/circle/

  1. Integrate with Plaid Link, which allows your customer to log into their bank and return you a public_token.

  2. Use Plaid's API to exchange the public_token for an access_token.

  3. Use the access_token with Plaid's API to create a Circle-specific processor_token.
