XeniApp : Xeniapp- Backend API - Integration Process

Created by User d8ec2, last modified on Jun 09, 2021

General Process for creating API. Here is the method to be followed while creating the rest API in nodejs.

  • Step 1: Define the route for the End point - API

    • Define Get/Post/Put/Delete

    • Define the url for the API

    • Create and call the method name for the API

Example : bhtpRouter.post('/bhtpQuote', bhtpQuotecall);

  • Step 2: Define the controller for the API

    • login into BHTP authentication API and get the token

    • Call the below service method

  • Step 3: Create the service for the API

    • call the BHTP third party API by sending the request

    • Receive the response if it is success return the response as status 200

    • If response fails return response 400 or 500
