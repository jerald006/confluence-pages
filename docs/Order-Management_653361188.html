XeniApp : Order-Management

Created by Gerardo Nizetich, last modified on May 24, 2024

Description

The Order-management is grouping all the services related to action/transaction based on booked items.
In the case of flight services, they are related to the PNR created that holds a flight itinerary.

  • Retrieve an Order

  • Order ticket (Issue the tickets and process the payment)

  • Change itinerary

  • Cancel the itinerary fully or partial

  • Issue EMD docs

  • Reshop (post-booking)

  • Reissue. (post-booking)

The Post-booking actions are related to a Ticketed PNR that need to be changed from the consumer.
Reshop : Is the process of Shopping based on a given itinerary with certain criteria. Offer the possible replace segments ticketed and calculate the additional to pay for fare difference and penalties.

Reissue: Is the Exchange ticket process, using the original paid ticket, based on a reshop reprice and issue a new ticket.
