Login to XeniApp

How to login to XeniApp?

Login to XeniPay

To perform any action on XeniPay, the user will need to login first.

Route: /user/login

Body:

{
    "email": "vivek@xeniapp.com",
    "password": "password"
}

Create Booking

Booking is created on the XeniApp platform.

Settle Booking

After a booking has been created, it can be settled. A booking can only be settled by the OTA that created it.

Route: /booking/settle

Body:

{
    "_id": "942de26a-4a32-6790342222df6"
}

Headers:

  • Authorization - Token returned from Login.

  • apiKey - OTA’s API key created during registration.

  • secretKey - OTA’s Secret key created during registration.

Redeem Booking

A booking can be redeemed only on XeniPay. A booking can only be redeemed after it has been in cooldown for 24 hours. However, for testing, this feature has been disabled.

Route: /booking/redeem

Body:

{
    "bookings": [181]
}

An array of booking IDs which can be retrieved from Create Booking.

Headers: Authorization which is the Token returned from Login.
