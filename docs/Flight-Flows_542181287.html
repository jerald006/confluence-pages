Search

 

Xeni facilitates end users to search for determined flight options based on filters like Trip type, Number of travelers, Cabin type. As per filters chosen by end user, the available flight options will be displayed.

User can search for flights by selecting filters as wanted, entering boarding point, destination and departure date.

 

When user selects one way trip, below fields have be filled before searching

o   Boarding Point

o   Destination

o   Departure Date

When user selects round trip, below fields have be filled before searching

o   Boarding Point

o   Destination

o   Departure Date

o   Return Date

When user selects multi-city trip, below fields have be filled before searching

o   Boarding Point

o   Destination

o   Departure Date

o   Add another flight

 

View Search details

 

Upon searching with required details, user is shown with results. The result page further provides user to filter out flights based on number of stops the flight has, departure and arrival times, Cabin Types, name of the Airlines, Cheapest flights, Best flights, Quickest flights.

User can select desired flight with these options and check the details of flight by clicking on View Details button

Book Now

Payment can be done using two methods

XeniPay for paying with cryptocurrencies

  1. Select XeniPay radio button and proceed with Complete Booking

  2. user has to select respective wallet and cryptocurrency

  3. user has to confirm the transaction from wallet

  4. Upon successful completion of transaction, user will be routed to billing page saying ”Processing Your Payment”. Same details can be found in user’s My Trips page with status “Awaiting Payment“. If transaction fails or session expires then status will be “Rejected”.

  5. Once payment is successful, status of the booking with changes to “Ticket Requested” automatically and booking details are emailed to user.

  6. Generation of ticket details will make the booking status to “Ticketed”.

Debit/Credit Card for paying with fiat currencies.

  1. Select Debit/Credit radio button and proceed with Complete Booking

  2. User has to select respective account details and confirm the payment.

  3. Upon successful completion of transaction, user will be routed to billing page saying ”Your booking has been confirmed”. Same details can be found in user’s My Trips page with status “Booked“. If transaction fails or session expires then status will be “Rejected”.

  4. Agency owner can notice “Ticket“ button for the Booked tickets.

  5. Agency Owner has to click ticket button to generate ticket details, this makes booking status to “Ticket Requested“.

  6. Ticket details are generated and emailed to user by agency owner, booking status is changed to “Ticketed' automatically.
