End User Car Rental Booking

From landing page, Car rentals can be made by navigating to Book Now to Cars tab.

Search Cars

  • User can search cars by

    • Selecting location, drop-off and entering drivers age.

      • for same location drop-off, user need to select pickup location, pickup time and drop-off time.

      • for different drop-off location, user need to select pick location, drop-off location, pickup time and drop-off time.

    • User can select filters as per needs and click on search button to get available cars deals

Car Deals

  • After entering details, clicking Search button results available car deals and deal information as per search criteria in a new page.

  • user has below other filter options to choose properties as per needs

    • Capacity

      • Passengers

      • Bags

    • Policies

    • Car type

    • Rental agency

    • Total price

    • Car features

  • User can filter properties using price filter

  • Clicking on View Deal button of a particular car deal routes to respective car deal page, where deal description, car accessories, pickup and drop off time, fuel policy, included mileage, estimated deposit, collision damage waiver, theft protection, price details are mentioned.

Book Now

  • After reviewing deal info and user can book deal by clicking on Book now button

  • Clicking Book Now routes to billing page, where preview of cancellation policy, car deal information is shown along with payment details and driver information which has to be entered by user.

Payment can be done using two methods

XeniPay for paying with cryptocurrencies

  1. Select XeniPay radio button and proceed with Complete Booking

  2. User has to select respective wallet and cryptocurrency.

  3. User has to confirm the transaction from wallet.

  4. Upon successful completion of transaction, user will be routed to billing page saying ”Processing Your Payment”. Same details can be found in user’s My Trips page with status “Awaiting Payment“. If transaction fails or session expires then status will be “Failed”.

  5. Once payment is successful, status of the booking with changes to “Confirmed” automatically. Same details can be found on user’s My Trips page for the booking with the Confirmation Booking ID.

  6. User can cancel the booking by clicking Cancel Booking from the action button of the booking. This changes the status of the booking to “Cancelled”

Debit/Credit Card for paying with fiat currencies.

  1. Select Debit/Credit radio button and proceed with Complete Booking

  2. User has to select respective account details and confirm the payment.

  3. Once payment is successful, status of the booking with changes to “Confirmed” automatically. Same details can be found on user’s My Trips page for the booking with the Confirmation Booking ID.

  4. User can cancel the booking by clicking Cancel Booking from the action button of the booking. This changes the status of the booking to “Cancelled”
