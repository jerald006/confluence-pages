XeniApp : Default Currency Feature

Created by vaibhav (Unlicensed) on Jan 12, 2023

  1. Overview:

    1. This feature will allow agency users to select their default currency, and all prices displayed within the platform will be automatically converted to that currency. Additionally, this feature will also enable agency users to process all payments in the selected default currency, and display the commission in the same currency as the payment was made, Super Admin should be able to see the default currency of each agency, and every report should include a column which specifies the currency in which the booking was made.

    2. This feature can also be extended to display the currency IP based. Agency owners will be able to decide whether to use the default currency or IP-based currency. Also, an agency owner can select currency at the user level as well.

    3. Currently, we have https://api.exchangeratesapi.io/v1/latest integrated.

  2. User Stories:

  • As an agency owner, I want to be able to select my default currency so that all prices displayed within the platform are in a currency I am familiar with.

  • As an agency owner, I want to be able to change my default currency at any time so that I can easily switch between currencies.

  • As an agency owner, I want to be able to process all payments in my default currency so that I don't have to worry about currency conversion fees.

  • As an agency owner, I want to see the commission in the same currency as the payment was made, so it's easy for me to compare the commission with the payment

  • As a super admin, I want to be able to see the default currency of each agency, so I can monitor and review & change the currency used across the platform.

  • As an agency user, I want to see the currency used in every report, so it's easy for me to understand in which currency the booking was made

3. User Flows:

User Flow for Agency Owner:

  1. The agency owner logs into the platform.

  2. The agency owner navigates to the settings page.

  3. The agency owner selects the default currency from a drop-down list of available currencies.

  4. The platform automatically converts all prices displayed within the platform to the selected default currency.

  5. Agency owner can change their default currency at any time by repeating the process.

  6. Agency owner can view commission in the same currency as the payment was made, so it's easy for them to compare the commission with the payment.

User Flow for Super Admin:

  1. Super admin logs into the platform.

  2. Super admin navigates to the admin panel.

  3. Super admin can view the default currency of each agency.

  4. Super admin can monitor and review the currency used across the platform.

  5. Super admin can also generate reports which include a column specifying the currency in which the booking was made for all the agencies.

  6. Super admin can also monitor the accuracy and reliability of IP-based currency detection.

4. Acceptance Criteria

  • Agency users can successfully select their default currency from a drop-down list of available currencies.

  • All prices displayed within the platform are automatically converted to the selected default currency.

  • Agency users can change their default currency at any time.

  • Prices are consistently displayed in the default currency throughout the platform

  • Agency users can process all payments in the selected default currency.

  • The commission is displayed in the same currency as the payment was made.

  • The super admin can view the default currency of each agency from the admin panel.

  • Every report generated by the system includes a column specifying the currency in which the booking was made.

  • The currency conversion is done in real time and reflects the current exchange rate.

  • The feature has minimal impact on the platform's overall performance.

  • The feature is compliant with all necessary regulations and laws related to currency conversion and payments.
