XeniApp : Signup Requirement

Created by User b60aa on Sep 20, 2021

User Should see the Signup pop-up by clicking on the Signup link in the Homepage or the Signup link in the Login pop-up.

The Signup popup should contain

  • Title as Signup

  • Button to close the pop-up.

  • Text box to enter the Name.

  • Email text box to enter the email.

  • Password field to enter the password.

  • Button to submit signup request.

  • A way to Signup by using Google and Facebook.

  • link to open the Sign-in pop-up.

Functionalities:

  • Successfully message should be shown after signing up with valid details.

  • The user needs to verify the email in order to log in to the application with signup details.
