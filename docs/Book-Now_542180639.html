XeniApp : Book Now

Created by Dheeraj Naithani (Unlicensed), last modified on Sep 07, 2021

For the book now feature we need to have deal with the following AWS services

  1. Route53 - To register domain . Setting up DNS lookup etc

  2. ACM - Create new certificate for the new domain or generate new certificate with all domains

  3. Cloud Front - create new cloud front distribution and associate new certificate with it or associate the all domains certificate with existing Cloud Front distribution

Scenario’s

  1. New domain registration for OTA

  2. OTA already has a domain and make Route53 DNS service for existing domain

  3. Registration OTA as subdomain of Xeniapp
