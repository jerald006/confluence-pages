XeniApp : User Login Requirement

Created by User b60aa, last modified on Sep 20, 2021

On the Homepage, the user should see a login popup after clicking on the Login button.

The Login pop view should contain

  • Title as Login

  • Email text box to enter the email.

  • Password text box to enter the password.

  • Button to submit the login request.

  • Link or button for forgot password request

  • A way to Login with Google and Facebook.

  • An option to signup.

  • Button to close the popup.

Functionalities:

  • For valid login, the user should navigate to the Booking page.

  • For Invalid login, the user should stay back on the Homepage.

  • Users should get noticed with a toast with a valid message for both valid and invalid logins.
