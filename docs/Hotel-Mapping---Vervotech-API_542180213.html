Search Component

Features

  1. Search Library

  • Ability to search by City - Compulsory

  • Ability to search by Hotel Name - Compulsory

  • Ability to search by Other Locations - Good to Have

  • API based on Geolocations

2. Checkin - Checkout Dates - Compulsory

  • Dates not allowed in the past (grey out past dates)

  • Check-out > Check-in

3. Rooms

  • Ability to add rooms with passengers - Compulsory

  • Ability to add multiple rooms - Compulsory

  • Ability to add children with age - Compulsory

  • Default should be 1 room, 2 Guests

4. Search CTA - Should get active only when all field are entered

5. View on MAP CTA - Should get active only when search results have appeared

6. Reset - Reset’s the last search
