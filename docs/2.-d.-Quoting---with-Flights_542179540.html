XeniApp : 2. d. Quoting - with Flights

Created by User d8ec2 on Jun 09, 2021

AirCare products require flights for purchase and for full quoting. to supply flights, use the "flights" array node in the request.

Sample Request with Flights for AirCare

Location

POST /v1/quote

Headers

Content-Type: application/json
Authorization: Bearer {your-token.from-the-authentication.response-here}

Request
