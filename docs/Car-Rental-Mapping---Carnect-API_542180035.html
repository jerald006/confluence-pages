Search Page

Search Component

Features

  • Ability to search by Pickup location - Mandatory

  • Ability to search by Drop-off location - Not Mandatory unless [Different drop-off] is selected

  • Ability to search by from date - Mandatory

  • Ability to search by to date - Mandatory

  • API based on Geolocations - Airports, Addresses and Cities

  • Ability to change country of residence location - Default to United States but list all countries

  • Ability to change to Different drop-off location - Default to Same drop-off location

  • Ability to change the driver age range - Default to 18-25

  • Search CTA - Validate mandatory fields and get results
