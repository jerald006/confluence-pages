XeniApp : Eligibility

Created by User d8ec2, last modified on Jun 09, 2021

You may also check the quote for validity and eligibility. This call works the same as a Full Quote but it differs in the fact that it does not process product information making the request faster and the response smaller.

Sample Request and Response

Location

POST /v1/eligibility

Headers

Content-Type: application/json
Authorization: Bearer {your-token.from-the-authentication.response-here}
