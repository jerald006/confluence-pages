Introduction

Create a new business line in XENIAPP to offer cruise product to consumers

Purpose

Increase the offer of functionalities and product into the XENI APP to fulfill the expectation of the customers and create a differentiator in the market.

Scope

MPV1

  • Create the Business Line “CRUISE”

  • Generate the backoffice for Agencies to manage the Cruise product

    • Markup

    • activity report

    • booking generated

    • bookings confirmed.

  • Implement the cruise integration via vendor API

  • Implement the Front end for users

  • Integrate the payment process for cruise product

  • Create metrics of use

  • Monitor activity and Vendor API performance.
