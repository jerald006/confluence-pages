XeniApp : Confirm the Edit

Created by User d8ec2 on Jun 09, 2021

Use this endpoint to pass the edit, along with payment information to the BHTP API to confirm the change. This runs the same validation and eligibility checks as the EditCheck endpoint.

Location

POST /v1/policy/editconfirm

Headers

Content-Type: application/json
Authorization: Bearer {your-token.from-the-authentication.response-here}
