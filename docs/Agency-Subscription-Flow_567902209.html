XeniApp : Agency Subscription Flow

Created by Ayush Kumar on Jun 08, 2023

Agency subscription flow is divided into two steps forms:

  1. First Steps form-> In this step, we will create an agency and user which will associate with that agency and user will treat as an Agency Owner

    1. POST → /api/agency

b. Flow Diagram:-

2. On second steps form:- In this steps user will choose the duration, enter card details and company address.

And These steps will act by two API. First API will verify the card and save the payload to respected collection.

POST → /api/verify/card

Flow Diagram:-

In second API, we will create subscription and get the details from DB for subscription

POST-> /api/agency/subscription
