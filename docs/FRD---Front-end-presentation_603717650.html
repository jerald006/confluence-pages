Search Widget

Widget components

  • Select Region: Displays the list of Regions according to the regions file from provider.

  • When: User can select my desired departure date and month

  • Duration: User could select an duration option like: (Display the name and return the value)

    • 1 for 1-4 nights,

    • 2 for 5-10,

    • 3 for 11-16,

    • 4 for 17-31,

    • 5 for 32+

  • Cruise Type: Displays the list of enabled cruises for Xeni from static Data.

  • Passengers: User can select the quantity and type of passengers as follow:

    • Adults: From 17 Yrs to end (Default value 2)

    • Children: From 2yrs to 17yrs (Default value: 0)

    • Infants: 0-2yrs (Default value: 0)

Non of the fields are required but
