Functional Requirements

Requirement

1

Get Expedia availability

2

Create RPC to match the HotelSearch to the expedia endpoint: /v3/properties/availability

3

Crete the RPC to Match the Expedia Content + Availability response with Xeni HotelSearch response.

4

Hotel Detail, do we need to create a new endpoint to fulfill the needs from the UI?

5

Offer Decorator: We need to identify the offer from 2 vendors of the same Hotel instance with the same conditions but different prices. Keep the lowest and detach the highest.

--header 'Customer-Ip: 5.5.5.5' \
