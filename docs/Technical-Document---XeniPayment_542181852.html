Xenipay

XeniPay is a simple, scalable, and easy-to-use Decentralized application that settles and validates transactions in the online travel industry.

Xeni-payment consists of three modules:

Xenipayment - gateway

When request (HTTP request) is made from Xeni, It is recieved in Xenipayment gateway module which performs initial checks and redirects (RPC calls) the requests to corresponding Bitpay or connexpay module.

Xenipayment - bitpay

This module is the implementation of Bitpay which is a bitcoin payment service provider.

Xenipayment - connexpay

This module is the implementation of ConnexPay which is a payments technology company that aligns card acceptance from customers with virtual card issuance to suppliers.
