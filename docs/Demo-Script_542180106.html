Step 2 - Edit Setting

  1. Sign In using the credentials

  2. Click on Settings - Edit Website

  3. Upload Logo - PNG FILE

  4. Upload Background Image

  5. Change Colors

  6. Change heading & subheading and publish

  7. Show the reflected changes on the Agency Page

  8. Click on Book Now - Search Flight & Hotel and add to cart

  9. Click on Bookings and show past trip toggle

Step 3 - Traveler Sign up

  1. Sign up a traveler

  2. Verify Email. Once the verification is done, remove userid=xyz from the URL and reload the url

  3. Login to the portal using the credentials.

  4. Click on Dashboard and show various options
