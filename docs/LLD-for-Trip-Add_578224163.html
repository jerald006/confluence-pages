XeniApp : LLD for Trip Add

Created by sonali, last modified on Oct 09, 2023

  1. On flight checkout page, when system gets flight details from flight API, system will call Create bundle offer API to get bundle products details with prices.

    1. POST → /api/createBundleOffer

  2. System will send this bundle product array to Trip add widget to display it on checkout page

  3. We are getting 2 prices array from Trip add API response. As per Trip add documentation, If user select whole bundle then bundled_pricing prices will get applied or if user selects specific product then prices in pricing array will get applied.

  4. On checkout page:

    1. User will fill the flight checkout form and select the product from bundle

    2. Once flight will get booked successfully, system will call Tripadd create order API with selected product IDs from bundle and passenger(lead passenger) details to create order

    3. POST → /api/createOrder

    4. After successful order creation, system will store orderId, the agency and platform commission of Trip add against booking in DB.

  5. Commission calculation:

    1. We are getting net_price and price from vendor. System will subtract net_price from price and calculate agency and platform markup on this remaining price
